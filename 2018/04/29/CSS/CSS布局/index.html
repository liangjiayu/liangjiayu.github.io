CSS布局

单页面布局

  • 适合用于简单的单页面布局
  • 场景 h5简单推广页 只用于手机端
  • container 表示单页面的背景
  • 内容需要固定宽度和高度居中
  • 单位vw、vh、rem做响应式单位

inline-block 两端布局

  • justify 表示一行的内联元素两端对齐
  • 这一行要填满一行，所以未行不会对齐

图片自适应

  • 原理通过子元素撑开父元素盒子
  • 适用于不同大小的图片的自适应 做响应式
